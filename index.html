Hello! I'm Kaindra Rizq Sachio, a 4th semester Computer Science student at Universitas Indonesia. I will be publishing my personal projects about Data Science & Artificial Intelligence here.

Projects

  • Efficient Fine-Tuning and Inference of LLMs under Resource Constraints for English–Indonesian Translation
    This project explores efficient fine-tuning and inference techniques for Large Language Models (LLMs) under limited resources, focusing on PEFT methods such as LoRA and QLoRA, and 4-bit inference using Qwen2.5 models for English–Indonesian translation. Blog

  • LLM Agent for News Search & Report Generation
    A fully automated agent that searches real-time news and generates structured reports based on user-defined topics. Built with LangChain and LangGraph, and powered by DeepSeek and Tavily. Demo

  • Indonesia Satellite Image Segmentation
    A web app for segmenting Indonesian satellite imagery into land classes like vegetation, infrastructure, and water using a U-Net model with a VGG16 encoder. Built with transfer learning on a custom-annotated dataset and visualized interactively.
    Demo

  • Interpretable Network Threat Detection using CatBoost and SHAP
    A web-based system for detecting and explaining network threats using a CatBoost classifier and SHAP values. Users can interactively explore local and global model explanations, provide feedback on predictions, and understand the influence of features on classification outcomes across multiple attack classes.
